📢 Marketing Campaign Performance Analysis
Analysis of a public marketing dataset using Python — covering customer 
segmentation, channel effectiveness, and ROI trends across 18 visualizations.

📌 Project Overview
This project evaluates marketing campaign performance across 5 key analytical 
dimensions using Python in Google Colab. It helps identify which customer 
segments respond best, which channels drive the most ROI, and where campaign 
spend can be optimized.

🛠️ Tools & Technologies

| Category      | Tools Used                          |
|---------------|-------------------------------------|
| Language      | Python 3                            |
| Libraries     | Pandas, Matplotlib, Seaborn         |
| IDE           | Google Colab                        |
| Dataset       | Public Marketing Campaign Dataset   |

🔍 Analysis Areas
1. Customer Segmentation — Age, income, and spending behaviour
2. Channel Effectiveness — Web, store, catalogue, deals comparison
3. Campaign Response Analysis — Which campaigns performed best
4. Product Performance — Revenue by product category
5. ROI Trends — Spend vs. revenue patterns

📊 Visualizations
Built 18 visualizations including bar charts, histograms, heatmaps, 
scatter plots, and pie charts for comprehensive business insight presentation.

🚀 How to Run
1. Open the notebook in [Google Colab](https://colab.research.google.com/)
2. Run all cells sequentially
